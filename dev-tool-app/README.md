# Developer Tool Playground

This project is a developer tool playground that allows users to write and execute code in a live environment. It features a code editor, live preview, theme switching, and shareable links for collaborative coding.

## Features

- **Monaco Editor**: A powerful code editor that supports syntax highlighting and IntelliSense for various programming languages.
- **Live Preview**: Instantly see the output of your code in a live preview pane.
- **Split Layout**: A user-friendly interface that allows you to view the code editor and live preview side by side.
- **Theme Switching**: Easily switch between different themes to customize your coding environment.
- **Shareable Link**: Generate a link to share your current code state with others.
- **Accessibility**: Navigate through the application using keyboard shortcuts for improved accessibility.

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm (Node Package Manager)

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/dev-tool-app.git
   ```
2. Navigate to the project directory:
   ```
   cd dev-tool-app
   ```
3. Install the dependencies:
   ```
   npm install
   ```

### Running the Application

To start the development server, run:
```
npm run dev
```
Open your browser and navigate to `http://localhost:3000` to access the application.

## Usage

- Use the Monaco editor to write your code.
- The live preview will automatically update as you type.
- Switch themes using the theme switcher component.
- Generate a shareable link to share your code with others.
